# AI Resume Analyzer

AI Resume Analyzer is a Flask-based web application that analyzes resumes and provides ATS compatibility scores, skill detection, resume evaluation, and job role prediction.

## Features

- Resume PDF Upload
- Resume Text Extraction
- Skill Detection
- ATS Compatibility Score
- Resume Scoring System
- Predicted Job Role
- Suggestions for Improvement
- Resume History Dashboard
- Login & Signup Authentication
- Session Management
- Chart Visualization using Chart.js

## Technologies Used

- Python
- Flask
- HTML
- CSS
- SQLite
- Chart.js
- pdfplumber

## How to Run the Project

### 1. Clone Repository

git clone https://github.com/Rizwana174/AI-Resume-Analyzer.git

### 2. Open Project Folder

cd AI-Resume-Analyzer

### 3. Install Required Libraries

pip install -r requirements.txt

### 4. Run Flask App

python app.py

### 5. Open Browser

http://127.0.0.1:5000
